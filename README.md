## Interview Questions

### What is OSI?

The OSI Model is a conceptual framework that explains how data travels across a network using seven layers.

### Which layer uses IP?

Layer 3

### Which layer uses MAC Address?

Layer 2

### Which layer uses TCP and UDP?

Layer 4

---

## Key Takeaways

- 7 Layers
- Layer 3 uses IP
- Layer 2 uses MAC
- Layer 4 uses TCP/UDP
